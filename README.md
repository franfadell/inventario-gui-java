# Inventory Management System – Java (Swing + POO)

Aplicación de gestión de inventario desarrollada en Java, utilizando Swing para la interfaz gráfica y POO.

## Características principales
- Interfaz gráfica con Swing  
- Manejo de productos (CRUD)  
- Carga y visualización de imágenes  
- Uso del patrón Singleton  
- Rutas universales compatibles en cualquier sistema operativo

## Tecnologías utilizadas
- Java 8+
- Swing
- POO

## Cómo ejecutar el proyecto

### Compilar
```
javac *.java
```

### Ejecutar
```
java Main
```
